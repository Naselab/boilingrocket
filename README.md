# 🚀 Boiling Rocket - Film Vault

This is the custom-coded portfolio for **Boiling Rocket**, featuring 40+ AI-driven short films and visual experiments.

## 🛠 Tech Stack
- **Frontend:** HTML5/CSS3 (Designed with Claude AI)
- **Hosting:** Netlify
- **Video Hosting:** Bunny.net Stream
- **Domain:** Porkbun (boilingrocket.com)

## 📁 Project Structure
- `index.html`: The main portal for all films.
- `/css`: Custom styles for the "Banned" aesthetic.
- `/assets`: Logos and thumbnails.

- Videos are embedded via Bunny.net to save on Netlify bandwidth.
